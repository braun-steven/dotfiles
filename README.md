# Dotfiles

![Screenshot](./scrot.png)

Run the following in `$dotfiles` to setup all symlinks:

```bash
./create-symlinks.py
```

The dotfiles contain configurations for:
* bash
* compton
* ctags
* dunst
* emacs
* flake
* i3
* i3status
* polybar
* pylint
* rofi
* termite
* tmux
* vim
* zathura
* zsh
